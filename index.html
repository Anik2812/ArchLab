Interactive Computer Architecture Simulator

User Manual

How to use the simulator:

  1. Drag components from the Component Shop to the Assembly Area.
  2. Click on components to learn more about them.
  3. Adjust the Clock Speed and Memory Capacity sliders.
  4. Enter instructions in the text area (one per line).
  5. Click "Run Simulation" to see the results.

Example Instructions:

  • calculate 2 + 2
  • read file from storage
  • write data to memory
  • render 3D graphics
  • send data over network
